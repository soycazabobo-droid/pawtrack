# 🐾 PawTrack — Rastreador de Paseos para Perros

Una app web **gratuita** para rastrear los paseos de tu perro en tiempo real usando el GPS de tu celular.

## ✨ Funcionalidades

- 📍 **Rastreo GPS en tiempo real** — dibuja el recorrido en el mapa mientras caminas
- 🗺️ **Mapa interactivo** — powered by OpenStreetMap (100% gratis)
- 📏 **Distancia y tiempo** — estadísticas en vivo durante el paseo
- 🐶 **Perfil de tu perro** — nombre, raza y avatar personalizado
- 📋 **Historial de paseos** — guarda los últimos 20 paseos automáticamente
- 📱 **Funciona en el celular** — solo abre el navegador, ¡sin instalar nada!

## 🚀 Cómo usarla

1. Abre la app en tu celular
2. Permite el acceso al GPS
3. Presiona **"Iniciar Paseo"** 🦮
4. ¡Camina con tu perro y ve el recorrido en vivo!
5. Presiona **"Terminar Paseo"** cuando termines

## 🛠️ Tecnologías (todas gratuitas)

| Tecnología | Uso |
|---|---|
| HTML/CSS/JS | App frontend |
| [Leaflet.js](https://leafletjs.com/) | Mapas interactivos |
| [OpenStreetMap](https://www.openstreetmap.org/) | Tiles del mapa |
| GitHub Pages | Hosting gratuito |
| localStorage | Guardar historial localmente |

## 🌐 Deploy en GitHub Pages (gratis)

1. Sube los archivos a un repo en GitHub
2. Ve a **Settings → Pages**
3. Selecciona la rama `main` y carpeta `/root`
4. ¡Tu app estará en `https://tu-usuario.github.io/pawtrack/`!
